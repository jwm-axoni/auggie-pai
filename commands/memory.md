---
description: Memory operations (read, reflect, patterns)
argument-hint: "<operation>"
---

Perform a memory system operation.

Available operations:
- **read**: Read recent reflections from `~/.augment/MEMORY/LEARNING/reflections.jsonl`
- **corrections**: Read captured corrections from `~/.augment/MEMORY/LEARNING/corrections.jsonl`
- **patterns**: Read synthesized patterns from `~/.augment/MEMORY/LEARNING/patterns.md`
- **status**: Show current work state from `~/.augment/MEMORY/STATE/current-work.json`
- **reflect**: Write a manual reflection entry to reflections.jsonl

$ARGUMENTS
